## 🔧 配置调参规范

### 分层调参规则

#### 1. 日常微调（策划每周执行）

**可调整项**：
- `MAP_DIFFICULTY_CURVE` - 各地图怪物系数±5%
- `BASE_GOLD_DROP_MULT` - 金币掉落±3%
- `BONE_ENHANCE_BASE_COST` - 强化费用±10%

**禁止修改**：
- 任何数学公式（如 soulPower=120*level^1.55）
- 转生属性加成（固定10%）
- 经验产出倍率梯度

#### 2. 月度版本优化

**可调整项**：
- `EXP_MULTIPLIER_BY_PRESTIGE[4+]` - 4转后经验系数
- `HIGH_TIER_RING_LOCK_PRESTIGE` - 高阶魂环解锁转生数
- 新增活动临时倍率配置

#### 3. 活动临时倍率（自动回滚）

```kotlin
GlobalValueConfig.EventMultipliers.activateEvent(
    mult = 2.0f,      // 双倍掉落
    durationDays = 3  // 持续3天
)
// 活动结束后自动恢复1.0倍
```
